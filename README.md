# Tweaks

| Tweak | Description |
|---|---|
| [YouTube-X](https://github.com/PoomSmart/YouTube-X) | A lightweight tweak with miscellaneous YouTube improvements and Adblock. |
| [YTSideload](https://github.com/Balackburn/YTSideload) | Minimal sideloading fix. Contains only the hooks required to make a resigned YouTube IPA run correctly. |
| [YouPiP](https://github.com/PoomSmart/YouPiP) | Enables native Picture-in-Picture for videos in the iOS YouTube app. |
| [YouQuality](https://github.com/PoomSmart/YouQuality) | Adds a video quality button to the video overlay for quick quality changes. |
| [YTVideoOverlay](https://github.com/PoomSmart/YTVideoOverlay) | Helper tweak that adds extra buttons to YouTube's video player overlay. |
| [YouMute](https://github.com/PoomSmart/YouMute) | Adds a mute/unmute button directly to the video overlay. |
| [Return YouTube Dislikes](https://github.com/PoomSmart/Return-YouTube-Dislikes) | Brings back the dislike count on YouTube videos using the Return YouTube Dislike API. |
| [YTABConfig](https://github.com/PoomSmart/YTABConfig) | Lets you configure A/B experiment flags in the YouTube app to unlock hidden features. |
| [YouGroupSettings](https://github.com/PoomSmart/YouGroupSettings) | Helper tweak that groups settings from multiple PoomSmart tweaks into one unified settings page. |
| [DontEatMyContent](https://github.com/therealFoxster/DontEatMyContent) | Prevents the notch/Dynamic Island from overlapping 2:1 video content in YouTube. |
| [NoYTPremium](https://github.com/PoomSmart/NoYTPremium) | Removes YouTube Premium upsell alerts and promotions. |
| [YTVideoErrorAlert](https://github.com/PoomSmart/YTVideoErrorAlert) | Shows an alert with the exact error details when a video fails to play. |
| [YouSpeed](https://github.com/PoomSmart/YouSpeed) | Adds a playback speed button directly to the video overlay for faster access. |
| [YTUHD](https://github.com/Tonwalter888/YTUHD) | Unlocks 1440p (2K) and 2160p (4K) resolution options in the YouTube app and it fixes playback. |
| [iSponsorBlock](https://github.com/Galactic-Dev/iSponsorBlock) | Implements the SponsorBlock API to automatically skip sponsored segments in YouTube videos. |
| [Alderis](https://github.com/hbang/Alderis) | A color picker library required by iSponsorBlock for its color customization UI. |
| [Open in YouTube](https://github.com/BillyCurtis/OpenYouTubeSafariExtension) | Adds an "Open in YouTube" prompt in Safari for sideloaded YouTube. |

# How to Build


- Fork this repository using the fork button on GitHub.
- Go to your profile and forked repositories, then go to Repository Settings/Actions and enable Read and Write permissions.</br>
 (Remember to always Sync the fork to the latest commit by clicking "Sync Fork" then "Update branch" on your forked page.)
- Go to Action Page
- Click "Build YTeto" on the left of the page
- Click "Run workflow" on the right of the page
- In the new opened pop-up, in the "URL to decrypted YouTube IPA", you need to provide the Decrypted Youtube.ipa file. </br>
  (You can get one from [EeveeDecryptBot](https://t.me/eeveedecrypterbot) on Telegram, [ArmConverter](https://armconverter.com/decryptedappstore/us) or [Decrypt.day](https://decrypt.day/)) </br>

- Once you have it, upload it somewhere using sites like [Dropbox](https://www.dropbox.com/) or [catbox.moe](https://catbox.moe/) </br>
 If you use Dropbox, change `dl=0` to `dl=1` at the end of the link or the download will fail. </br>
 
- Now select all the tweaks you want to enable or disable (all of them are enabled by default)
- Then click "Run Workflow"
- Wait until the workflow finishes
- You will find YTeto.ipa in the release tab of your forked page.

- Download the .ipa and Install it with [Feather](https://github.com/khcrysalis/Feather) if you have a certificate or [AltStore](https://altstore.io/), [Sidestore](https://sidestore.io/) and [LiveContainer](https://github.com/LiveContainer/LiveContainer) otherwise.

- Enjoy!
